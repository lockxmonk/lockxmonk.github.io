Filter

2017/4/13 posted in  Python练习

1.请尝试用filter()删除1~100的素数。

# -*- coding: utf-8 -*-
num = range(1, 101)

def prime_num(s):
    for x in range(2, s):
        if (s % x) == 0:
            return True
        else:
            return False
print filter(prime_num, num)
